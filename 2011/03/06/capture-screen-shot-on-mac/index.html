Capture Screen Shot on Mac

Posted on March 6, 2011

To capture a screen shot on Mac:

Capture by region selection:
– press command + shift + 4
– you will see the mouse cursor change to cross-hairs
– define the area, by dragging mouse while your left mouse button is pressed
– release the mouse button to capture and save the image on desktop

Capture window
– press command + shift + 4
– you will see the mouse cursor change to cross-hairs
– press space-bar ( see the camera icon )
– click the mouse button to capture and save the image on desktop
