days between dates

Posted on June 8, 2014

Seems like finding out days between dates is not a trivial task. There is quite a bit of support in tools for adding up dates (date), but literally none for evaluating the difference. Unix systems return time using epoch as reference, thus you need to divide the difference by 86400 (#seconds in a day) to get the #days.
