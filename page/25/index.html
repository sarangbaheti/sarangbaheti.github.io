library, framework and bundle

November 27, 2011

library, frameworks, and bundles in Mac OS X are similar in that you can think of them as pre-packaged code meant to be used from within another library or application. They’re all meant to improve or expand the functionality of an executable (including those that are part of the operating system) by making their code available to it. library The term library has the same meaning on every platform. A library is the most basic form of prepackaged, shared code meant to be reused in multiple applications.
