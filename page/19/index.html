Posted on December 14, 2012

Earlier (pre Xcode 4.3) toolchains were installed in /usr/bin. But after Xcode has been bundled toolchains installation to /usr/bin is optional and are local to Xcode application. To install toolchain (clang & llvm) as to use them on command line go to Xcode preferences and Download and install. Or other option is to use xcrun
