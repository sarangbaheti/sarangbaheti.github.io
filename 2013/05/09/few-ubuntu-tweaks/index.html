few ubuntu tweaks

Posted on May 9, 2013

Ubuntu is a great OS to use, but there are few things that can make life a bit more easier. First and foremost I don’t like those flash overlaid scroll-bars. Open a terminal and execute this command:

sudo apt-get remove overlay-scrollbar

and result is pleasing, notice the normal scrollbars



Ubuntu these days is the poster boy of Linux for mere mortals, thus Canonical has cut down on installing build related tools by default. Get your machine up to date for build related stuff by executing this command

sudo apt-get install build-essentials flex bison gcc-multilib



Take control of privacy, with Ubuntu 13.04 search includes results from online sources as well. You can disable them here:



To take control of every nook and corner of your Ubuntu’s look and feel, install Unity Tweak tool:



Install jupiter to control the power mode, as in Windows:

sudo apt-add-repository ppa:alanbell/unity</br>
sudo apt-get update</br>
sudo apt-get install unity-window-quicklists



Install Weather Indicator right in your app-menu:

sudo add-apt-repository ppa:atareao/atareao</br>
sudo apt-get update</br>
sudo apt-get install my-weather-indicator



Get quick lists between open windows, much like jump-lists in Windows 7:

sudo add-apt-repository ppa:tualatrix/ppa</br>
sudo apt-get update</br>
sudo apt-get install ubuntu-tweak
